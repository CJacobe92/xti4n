# Mail Server with Postfix and Dovecot

In an effort to simulate internet infrastructure, I've built a self-contained environment that integrates both authoritative and recursive DNS servers. This virtual production system emulates a real-world network setup, allowing me to explore ethical hacking, advanced network security, and DNS architecture—all without relying on live production servers. To further enhance realism, we also need an SMTP server functioning like Google Mail to simulate real-world email scenarios.

### System Requirements

* CentOS with 2GB ram, 20GB HDD, 2 vCPU
